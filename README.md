<h1>YDKEN  (You Don't Know English)</h1>
<p>A simple React app using the "You Don't Know JS" books.</p>

<p>The idea is simple</p>
<ul>
 <li>Use information that really interests me.</li>
 <li>Avoid context switching</li>
</ul>
